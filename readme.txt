== Installation ==

This section describes how to install the plugin and get it working.

1. Delete any existing `bookmark` or `wp-best-social-bookmark` folder from the `/wp-content/plugins/` directory
2. Upload the entire content of plugin archive to your `/wp-content/plugins/` directory, 
   so that everything will remain in a /wp-content/plugins/wp-best-social-bookmark/ folder
2. Open the plugin configuration page, which is located under Options -> Best Social Bookmark Menu
3. Activate the plugin through the 'Plugins' menu in WordPress (deactivate and reactivate if you're upgrading).
4. Done. Enjoy.
